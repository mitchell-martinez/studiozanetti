# ProcessTimelineBlock Component

Step-by-step timeline block for showing the client journey from enquiry to delivery.

## WordPress Admin Setup

1. Edit a page in WordPress.
2. In **Content Blocks** add layout **process_timeline**.
3. Fill fields:
   - **Heading** (optional)
   - **Intro** (optional)
   - **Steps** (repeater)
     - **Title** (required)
     - **Description** (required)
     - **Image** (optional)
   - **Section Theme / Spacing** (optional styling controls)

## Notes

- Ideal for: first contact, planning, wedding day, post-production, delivery.
- Keep step titles short and outcome-focused.
