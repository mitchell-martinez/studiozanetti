# PricingPackagesBlock Component

Flexible package grid for photography pricing tiers and inclusions.

## WordPress Admin Setup

1. Edit a page in WordPress.
2. In **Content Blocks** add layout **pricing_packages**.
3. Fill fields:
   - **Heading** (optional)
   - **Subheading** (optional)
   - **Packages** (repeater)
     - **Name** (required)
     - **Price Label** (optional, e.g. `$1,980`)
     - **Description** (optional)
     - **Inclusions** (optional, WYSIWYG)
     - **Is Featured** (optional)
     - **CTA Text** (optional)
     - **CTA URL** (optional)
   - **Section Theme / Spacing** (optional styling controls)

## Notes

- Use `Is Featured` to highlight your most popular package.
- Keep package names and pricing labels consistent for easy comparison.
