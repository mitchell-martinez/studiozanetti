# TestimonialCarouselBlock Component

Rotating testimonial slider for wedding/event client feedback.

## WordPress Admin Setup

1. Edit any page in WordPress.
2. In **Content Blocks** add layout **testimonial_carousel**.
3. Fill fields:
   - **Heading** (optional)
   - **Subheading** (optional)
   - **Auto Rotate Seconds** (optional, default 6)
   - **Testimonials** (repeater)
     - **Quote** (required)
     - **Name** (required)
     - **Context** (optional, e.g. `Wedding`)
     - **Image** (optional)
   - **Section Theme / Spacing** (optional styling controls)

## Notes

- If only one testimonial is added, dots and auto-rotation are hidden.
- Use this for reviews, thank-you mail, and social proof sections.
