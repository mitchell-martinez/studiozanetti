# FaqAccordionBlock Component

Expandable FAQ accordion for common wedding and event questions.

## WordPress Admin Setup

1. Edit a page in WordPress.
2. In **Content Blocks** add layout **faq_accordion**.
3. Fill fields:
   - **Heading** (optional)
   - **Intro** (optional)
   - **Open First Item** (true/false)
   - **FAQ Items** (repeater)
     - **Question** (required)
     - **Answer** (required, WYSIWYG)
   - **Section Theme / Spacing** (optional styling controls)

## Notes

- Best used on `experience`, `pricing`, and `contact` pages.
- Add 5–10 concise questions for strongest conversion impact.
